The Wine Reviews Dataset

The Wine Reviews Dataset we worked with was taken from Kaggle. It contains 130,000 wine reviews with variety, location, winery, price, points, and description information on over 110,000 unique wine products.
The data was originally scraped from WineEnthusiast, a website containing various reviews and information about wine.
